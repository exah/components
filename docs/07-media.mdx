# Media

## Description

Implements [Media Object](http://www.stubbornella.org/content/2010/06/25/the-media-object-saves-hundreds-of-lines-of-code) with [`Flex`](/flex) - one of most useful layout components.

Components:

- `Media` — Container
- `Media.Content` — Expands to take available space
- `Media.Side` — Can be placed on left or right and take content size


## Usage

```js
import { Media } from 'pss-components'
```

```js
<Media>
  <Media.Side>
    Side
  </Media.Side>
  <Media.Content>
    Content
  </Media.Content>
</Media>
```
