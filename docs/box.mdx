# Box

Core layout component, that render `<div>` by default and used inside all components. See [box](https://github.com/exah/pss/blob/master/docs/api.md#box) in `pss` docs.

## Usage

```js
import { Box } from 'pss-components'
```

### Reset styles

By default it applies this reset styles:

```css
font: inherit;
color: inherit;
background: transparent;
border: 0;
margin: 0;
padding: 0;
```

You can override it in `theme.Box`:

```js
import { css } from '@emotion/core'

// only reset margin and padding, everything else is untouched
const myTheme = {
  Box: css`
    margin: 0;
    padding: 0;
  `
}
```
