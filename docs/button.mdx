# Button

Based on [Box](/box). Renders `<button>` element with reset styles:

```css
appearance: none;
textAlign: center;
textDecoration: none;
cursor: pointer;
[disabled] {
  cursor: auto;
  pointerEvents: none;
}
```


## Usage

```js
import { Button } from 'pss-components'
```

<Playground>
  <Button mr={1}>Button</Button>
  <Button variant='auto'>Button</Button>
  <Button bg='red' ml={1}>Button</Button>
</Playground>
