# Feed

Masonry-like grid component, created with [FlexGrid](/flex-grid) and requires [MatchMediaProvider](/match-media).

## Usage

```js
import { Feed } from 'pss-components'
```

```js
<MatchMediaProvider>
  <Feed space={2} grid={12} column={{ all: 3, md: 6, sm: 12 }}>
    <Feed.Item>
      Item
    </Feed.Item>
  </Feed>
</MatchMediaProvider>
```
