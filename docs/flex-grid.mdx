# FlexGrid

Grid component based on [Flex](/flex) and [Box](/box). 

Useful if you need to support old browsers without `display: grid`, 
if don't: use [Grid](/grid) instead.


### Components:

#### `FlexGrid` — Container component:

- `columns` - Number of columns in grid, default to `12`
- `space` — horizontal and vertical space
- `spacex` — horizontal space
- `spacey` — vertical space

For possible values for spacing see [space docs](https://github.com/exah/pss/blob/master/docs/api.md#space).


#### `FlexGrid.Item` — Item / column component, size can be set with props:

- `column` — Number in range `1...columns`
- `offset` — Number in range `1...columns` — move item by number of columns


## Usage

```js
import { FlexGrid } from 'pss-components'
```

```js
<FlexGrid columns={12}>
  <FlexGrid.Item column={6}>
    <Box>
      Content
    </Box>
  </FlexGrid.Item>
  <FlexGrid.Item column={4}>
    <Box>
      Content
    </Box>
  </FlexGrid.Item>
  <FlexGrid.Item column={2}>
    <Box>
      Content
    </Box>
  </FlexGrid.Item>
</FlexGrid>
```

### Create your own grid

```js
const MyGrid = (props) => (
  <FlexGrid
    columns={16}
    space={1}
    {...props}
  />
)
```
