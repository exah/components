# Flex

[Box](/box) with `display: flex` and additional `pss` styles:

- [flexContainer](https://github.com/exah/pss/blob/master/docs/api.md#flexcontainer) - `flexWrap`, `flexDirection`
- [boxContentAlignment](https://github.com/exah/pss/blob/master/docs/api.md#boxcontentalignment) - `justifyContent`, `alignContent`
- [boxItemsAlignment](https://github.com/exah/pss/blob/master/docs/api.md#boxitemsalignment) - `alignItems`, `justifyItems`
- [gap](https://github.com/exah/pss/blob/master/docs/api.md#gap) - `gap`, `rowGap`, `columnGap`


## Usage

```js
import { Flex, Box } from 'pss-components'
```
