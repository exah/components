# Flex

## Description

Based on [Box](/box) with [flexbox](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout) styles.


## Usage

```js
import { Flex, Box } from 'pss-components'
```

```js
<Flex>
  <Box>Item</Box>
</Flex>
```
