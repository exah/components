# Getting Started

## 1. Install

```sh
$ yarn add @emotion/core @emotion/styled pss-components
```

## 2. Import components

```js
import { Flex, Box, Text, Button, Image } from 'pss-components'
```

### Colors

[ðŸ“– Docs](https://github.com/exah/pss/blob/master/docs/api.md#colors)

```
const myTheme = {
  color: {
    black: '#000',
    grey: [ '#eee', '#ddd', '#ccc', '#bbb', '#aaa' ],
    white: '#fff'
  },
  palette: {
    default: {
      bg: '#fff', // default background color
      fg: '#000', // default text color
      primary: '#ff0',
      positive: '#0f0',
      negative: '#f00'
    }
  }
}
```

<Playground>

      <Box m={1} width={1 / 4} ratio={1} bg='auto' /* enable default */ />
    </Flex>
  </ThemeProvider>
</Playground>
