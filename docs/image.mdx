# Image

Based on [Box](/box) with `display: block` and `max-width: 100%`.


## Usage

```js
import { Image } from 'pss-components'
```

<Playground>
  <Image 
    src='http://placekitten.com/500'
    alt='Kitten looking down'
    maxWidth={1 / 2}
  />
</Playground>
