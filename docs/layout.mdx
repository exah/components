# Layout

Implements [Media Object](http://www.stubbornella.org/content/2010/06/25/the-media-object-saves-hundreds-of-lines-of-code) with [Flex](/flex) - one of most useful layout components.

Components:

- `Layout` — Container
- `Layout.Content` — Expands to take available space
- `Layout.Side` — Can be placed on left or right and take content size


## Usage

```js
import { Layout } from 'pss-components'
```

```js
<Layout>
  <Layout.Side>
    Side
  </Layout.Side>
  <Layout.Content>
    Content
  </Layout.Content>
</Layout>
```
