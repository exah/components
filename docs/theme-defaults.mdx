# ThemeDefaults

Change `theme.default` with props.


## Usage

```js
import { ThemeDefaults } from 'pss-components'
```

## Examples

<Playground>
  <Box>

    </Box>
    <ThemeDefaults palette='inverted'>

        Content
      </Box>
    </ThemeDefaults>
  </Box>
</Playground>
