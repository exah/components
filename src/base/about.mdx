# Base, createBaseComponent

## Description

Component without any styles, that filters unkown props. Underlying component can be changed with `comp` prop. By default renders `<div>`.


## Usage

```js
import { Base, createBaseComponent } from 'pss-components'
```

```js
<Base>
  {/* children */}
</Base>
```


```js
const Heading = createBaseComponent('h1')
```

```js
<Heading>
  {/* children */}
</Heading>
<Heading comp='h2'>
  {/* children */}
</Heading>
```
