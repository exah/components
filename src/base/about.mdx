# Base, createBase

## Description

Component without any styles, that filters unkown props. Underlying component can be changed with `as` prop. By default renders `<div>`.
  
For more info see [@exah/react-base-component](https://github.com/exah/react-base-component) docs.


## Usage

```js
import { Base, createBase } from 'pss-components'
```

```js
<Base>
  {/* children */}
</Base>
```


```js
const Heading = createBase('h1')
```

```js
<Heading>
  {/* children */}
</Heading>
<Heading as='h2'>
  {/* children */}
</Heading>
```
