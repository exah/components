# Box

## Description

Basic component, that renders as `<div>` with [system prop styles](https://github.com/exah/pss/blob/master/docs/api.md#system). Component inside can be changed with `as` prop.


## Usage

```js
import { Box } from 'pss-components'
```

```js
<Box>
  <Box>1</Box>
  <Box>2</Box>
</Box>
```
