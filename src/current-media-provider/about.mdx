# CurrentMediaProvider

## Description

Watch `media` defined in `theme` for changes and pass currently active media as `currentMediaKey` prop.


## Usage

```js
import { CurrentMediaProvider, CurrentMediaConsumer, withCurrentMedia } from 'pss-components'
```

```js
import { ThemeProvider } from 'emotion-theming'

<ThemeProvider theme={{ media: { M: '(max-width: 600px)' } }}>
  <CurrentMediaProvider>
    {/* components */}
  </CurrentMediaProvider>
</ThemeProvider>
```

```js
import { withCurrentMedia } from 'pss-components'

const MediaComponent = withCurrentMedia(Component)
```
