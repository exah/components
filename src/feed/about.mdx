# Feed

## Description

Masonry-like grid component, created with [Grid](#/grid).

## Usage

```js
import { Feed } from 'pss-components'
```

```js
<ThemeProvider theme={{ grid: 12 }}>
  <CurrentMediaProvider>
    <Feed space={2} col={3} colT={6} colM={12}>
      <Feed.Item>
        Item
      </Feed.Item>
    </Feed>
  </CurrentMediaProvider>
</ThemeProvider>
```
