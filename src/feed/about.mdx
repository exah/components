# Feed

## Description

Masonry-like grid component, created with [FlexGrid](#/flex-grid).

## Usage

```js
import { Feed } from 'pss-components'
```

```js
<ThemeProvider theme={{ grid: 12 }}>
  <CurrentMediaProvider>
    <Feed space={2} column={{ all: 3, md: 6, sm: 12 }}>
      <Feed.Item>
        Item
      </Feed.Item>
    </Feed>
  </CurrentMediaProvider>
</ThemeProvider>
```
