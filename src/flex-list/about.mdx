# FlexList

Based on based on [`FlexBox`](#/flex-box) with applied reset styles for `ul`, `ol`. For more information see `FlexBox` docs.

## Usage 

```js
import { FlexList } from 'pss-components'
```

```js
<FlexList>
  <FlexList.Item>Item</FlexList.Item>
</FlexList>
```
