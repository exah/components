# Grid

## Description

Grid component created with [`FlexBox`](#/flex-box).

- Default to `12` columns. Must be defined in `theme.grid`
- Item size can be set with `col` or `size` prop in `Grid.Item` component
- Space between items can be set with `space`, `spacex` and `spacey` props

Components:

- `Grid` — Container component. Control space between items with props:
  + Possible values can be set in `theme.space`, see [space docs](https://github.com/exah/pss/blob/master/docs/api.md#space)
  + `space` — setting horizontal and vertical space
  + `spacex` — setting horizontal space
  + `spacey` — setting vertical space
  + `spaceContent` — apply space to `Grid.Content` instead of `Grid.Item`
- `Grid.Item` — Item / column component, size can be set with props:
  + `col` — Number in range `1...theme.grid`, result is precentage size. (`col / theme.grid`, i.e. `3 / 12 = 25%`)
  + `size` — Number from `0...1`, result is percentage (`0.5` is `50%`)
- `Grid.Content` — Content wrapper same as [`Box`](#/box)

## Usage

```js
import { Grid } from 'pss-components'
```

```js
<ThemeProvider theme={{ grid: 12 }}>
  <Grid space={2}>
    <Grid.Item col={6}>
      <Grid.Content>
        Content
      </Grid.Content>
    </Grid.Item>
    <Grid.Item col={4}>
      <Grid.Content>
        Content
      </Grid.Content>
    </Grid.Item>
    <Grid.Item col={2}>
      <Grid.Content>
        Content
      </Grid.Content>
    </Grid.Item>
  </Grid>
</ThemeProvider>
```
