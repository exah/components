# Layout

## Description

Implements [Holy Grail Layout](https://en.wikipedia.org/wiki/Holy_grail_(web_design)) with [`FlexBox`](#/flex-box).

- Header is on top of page
- Footer is stick to "bottom" of page
- Content expands to take available space
- Body can be used as container to equal height columns (like navs) and main Content

Components:

- `Layout` — Container that takes space of page (add `minHt='100vh'` prop, default to `100%`)
- `Layout.Content` — Main component, that expands to take available space
- `Layout.Body` — Can be used inside `Layout` as main container of vertical columns and content

## Usage

```js
import { Layout, Box } from 'pss-components'
```

```js
<Layout>
  <Box>Top</Box>
  <Layout.Body>
    <Layout.Content>Content</Layout.Content>
    <Box>Side Nav</Box>
  </Layout.Body>
  <Box>Bottom</Box>
</Layout>
```
