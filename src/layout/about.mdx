# Layout Component

Implements [Holy Grail Layout](https://alistapart.com/article/holygrail) with [`FlexBox`](#/flex-box).

Key features:

- Header is on top of page
- Footer is stick to "bottom" of page
- Content is expands to take available space
- Body can be used as container to equal height columns (like navs) and main Content
