# MediaObject

## Description

Implements [Media Object](http://www.stubbornella.org/content/2010/06/25/the-media-object-saves-hundreds-of-lines-of-code) with [`FlexBox`](#/flex-box) - one of most useful layout components.

Components:

- `MediaObject` — Container
- `MediaObject.Content` — Expands to take available space
- `MediaObject.Side` — Can be placed on left or right and take content size


## Usage

```js
import { MediaObject } from 'pss-components'
```

```js
<MediaObject>
  <MediaObject.Side>
    Side
  </MediaObject.Side>
  <MediaObject.Content>
    Content
  </MediaObject.Content>
</MediaObject>
```
